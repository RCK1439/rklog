# rklog examples

## Compilation

- Run the following in the terminal for the *basic* example:

```bash
make basic
```

- And then for the *custom* example:

```bash
make custom
```

- Finally, to clean up any artifacts:

```bash
make clean
```

## Running

Running is as easy as

```bash
./basic_logger_example
```

or

```bash
./custom_logger_example
```
